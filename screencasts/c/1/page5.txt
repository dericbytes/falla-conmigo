




   ¡Empezamos! Vamos a crear un archivo. El nombre del archivo no
   importa. Pero si usamos la extension '.c'. Nuestro editor y otros
   programas pueden usar la extension para anotar el tipo de archivo
   y proporcianar un comportamiento más apropiado.

   Por ejemplo formatear el archivo en una manera más facíl para
   leer. O adivina unas acciones adecuadas, por ejemplo, el commando
   'compilar' podría funcionar sin la necesidad de decir cuál tipo
   de compilador a usar.
